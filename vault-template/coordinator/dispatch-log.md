# Coordinator Dispatch Log — <project>

Append-only log of dispatch decisions: authorizations, refusals, and
grilling-gate questions. Each entry is timestamped and includes rationale.

**Format per entry:**
- **Timestamp** (UTC, ISO 8601)
- **Action** (authorized / refused / grilling-question / pairing-check)
- **Target** (which role/domain/plan)
- **Rationale** (why)
- **Follow-up** (if any)
